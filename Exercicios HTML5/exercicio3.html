O que é linguagem de programação?

A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte - que pode ser compilado e transformado em um programa de computador, ou usado como script interpretado - que informará instruções de processamento ao computador.

Exemplos:

  • Python
  • Java
  • Java Script
  • C++
  • C
  • Go

Linguagem de programação Server Side

O que é?

As linguagens server-side são linguagens que o SERVIDOR (lado do servidor) entende, ou seja, aplicações que rodam no servidor. Isso quer dizer que você vai escrever um código onde o servidor vai processá-lo e então vai mandar para o seu navegador a resposta.

  • PHP
  • ASP
  • Java
  • Ruby
  • Python

Linguagem de programação Cliente Side

O que é?

A linguagem de cliente ou client-side scripting, é uma linguagem que é executada no lado cliente, ou seja, no computador do próprio usuário, e por isso é usada nas situações em que a linguagem server-side não tem alcance.

  • HTML
  • CSS
  • JavaScript
  • AJAX
