Introdução à Web Development

  • Aprender sobre a arquitetura da web
  • Compreender a diferença entre o front-end e o back-end
  • Aprender os conceitos básicos de HTML, CSS e JavaScript

HTML5

  • Aprender a sintaxe e as tags HTML
  • Aprender a estrutura básica de uma página HTML
  • Aprender a criar formulários
  • Aprender a criar links
  • Aprender a trabalhar com imagens, vídeos e áudios
  • Aprender a utilizar as tags semânticas do HTML5

CSS

  • Aprender a sintaxe e as propriedades CSS
  • Aprender a criar e aplicar estilos em elementos HTML
  • Aprender a trabalhar com seletores CSS
  • Aprender a trabalhar com layouts responsivos e design responsivo
  • Aprender a trabalhar com animações e transições CSS

JavaScript

  • Aprender a sintaxe e a estrutura básica do JavaScript
  • Aprender a utilizar variáveis e tipos de dados em JavaScript
  • Aprender a utilizar funções e métodos em JavaScript
  • Aprender a trabalhar com eventos em JavaScript
  • Aprender a trabalhar com DOM em JavaScript
  • Aprender a utilizar bibliotecas e frameworks JavaScript, como jQuery e React

Prática e Projetos

  • Criar pequenos projetos e exercícios práticos para consolidar o conhecimento adquirido em HTML, CSS e JavaScript
  • Participar de fóruns de desenvolvimento web para compartilhar e obter feedback sobre projetos pessoais e profissionais
  • Participar de desafios de codificação para aprimorar habilidades em desenvolvimento web.
